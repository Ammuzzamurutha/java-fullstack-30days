# Day 01 - Hello World Program ☕

## 📌 Objective

The objective of Day 1 is to understand the basic structure of a Java program and successfully run a simple "Hello World" application.

---

## 💻 Program Description

This program prints a simple message to the console:

Hello, World!

It demonstrates:
- Basic Java syntax
- Class structure
- `main()` method
- Use of `System.out.println()`

---

## 🛠️ Concepts Practiced

- Java program structure
- `public class`
- `public static void main(String[] args)`
- Console output
- Compilation and execution of Java program

---

## 📂 File Included

- `HelloWorld.java`

---

## 🚀 Learning Outcome

By completing this program, I learned:
- How Java programs are structured
- How execution starts from the `main()` method
- How to print output to the console
